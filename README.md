# Jokempo Go!

-  Lição de Desenvolvimento de Aplicações Móveis do 2º ano da Germinare Tech
- Aplicativo mobile simples de pedra, papel e tesoura

## Funcionalidades

- Escolher entre pedra, papel ou tesoura
- Placar dinâmico
- Cores de acordo com resultado
